Objetivo

Para esta práctica vamos a utilizar una máquina de Ubuntu que la asignaremos como cliente de dominio y la uniremos al PDC de Windows Server que hemos manejado en las practicas anteriores. Para ello necesitamos que en la máquina de Ubuntu instalemos LikeWise y lo configuremos mediante el pdf de dirac-01.

Configuracion de Clientes

Para conectar la máquina de Ubuntu al PDC tenemos que contar con la hora y fecha. Aparte, ambas máquinas tienen que estar en la misma red estática, así que las configuraremos por comandos en el archivo interfaces (nano /etc/network/interfaces), también deberíamos configurar los servidores DNS editando el archivo /etc/resolv.conf y después comprobando que todo se ha realizado bien.

Union al Dominio

Ya que vamos a unirnos mediante el cliente Ubuntu al PDC de Windows Server utilizando el LikeWise necesitaremos descargarlo de la web oficial, instalarlo y usando un terminal, ejecutaremos domainjoin-gui y rellenamos los que nos pide introduciento el dominio del PDC. Nos pedirá el usuario y la contraseña y despues, verificamos que el cliente de Ubuntu se ha añadió sin errores.
